# Weather Data Analysis using Python

## Overview
This project performs exploratory data analysis (EDA) on historical weather data using Python.

The goal of this project was to understand weather patterns, identify trends, and analyze relationships between different weather variables.

## Analysis Performed
- Data loading and quality checks
- Handling data types and date/time feature extraction
- Feature engineering:
  - Season classification
  - Temperature categories
  - Time-based features
- Temperature trend analysis
- Weather condition frequency analysis
- Visibility analysis
- Humidity and visibility relationship analysis
- Seasonal analysis of wind speed and pressure
- Temperature and dew point relationship analysis
- Correlation analysis between weather variables

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

## Key Insights
- Identified seasonal and hourly temperature patterns
- Observed variations in visibility under different weather conditions
- Analyzed how humidity levels affect visibility
- Compared weather characteristics across seasons
- Examined relationships between temperature, dew point, and other weather variables

## Conclusion
This project demonstrates the use of Python for data cleaning, exploratory analysis, visualization, and extracting meaningful insights from real-world weather data.
